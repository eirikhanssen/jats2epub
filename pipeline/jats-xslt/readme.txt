NISO Journal Article Tag Suite (JATS) version 1.0

This folder contains only the JATS preview stylesheets required for ePub production 
from JATS xml-tagged articles for 
the journal Professions & Professionalisms at Oslo and Akershus University College.



If your organization wants to create a similar workflow, 
you might find other xslt stylesheets in the original JATSPreviewStylesheets-package that you might need.


The XSLT stylesheets conained here, as well as other preview stylesheets developed for JATS can be found in the xslt folder 

in the NCBITools/JATSPreviewStylesheets package listed at the top of this readme.

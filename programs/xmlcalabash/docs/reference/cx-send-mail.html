XML Calabash Reference (Version 0.10)

Name

cx:send-mail — Sends an email message.

Synopsis

<p:declare-step type="cx:send-mail" xmlns:cx="http://xmlcalabash.com/ns/extensions">
     <p:input port="source" sequence="true"/>
     <p:output port="result"/>
</p:declare-step>

Description

The first document on the source port is expected to conform to An XML format for mail and other messages. Any additional documents are treated as attachments.

The em:content may contain either text or HTML. To send some other type as the first message body, you must leave the em:content element out of the first document and supply the body as a second document.

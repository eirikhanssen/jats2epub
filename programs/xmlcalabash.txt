xmlcalabash is used to by jats2epub to run the XProc pipeline.
The XProc pipeline does the heavy lifting and converts the input xml-file to various files needed to build the epub archive.

xmlcalabash is required by jats2epub, jats2epub will not be able to run without it.

please consult the install instructions in jats2epub/README matching your operating system for how to install xmlcalabash.
